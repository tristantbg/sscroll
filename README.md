# sscroll-easings
Itty-bitty smooth scrolling. **400 bytes gzipped.**

# Usage
```javascript
import sscroll from 'sscroll-easings'

const stop = sscroll(document.getElementById('target'), {
  duration: 500, // ms
  offset: 50 // positive values scroll further
})

stop() // stops scroll
```
